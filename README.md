# El Sitio del Blog - Grupo 2

## Descripción

Este proyecto consiste en el desarrollo de un sitio web de blog sobre Inteligencia Artificial (IA), elaborado como parte del Laboratorio de Desarrollo Web en Equipo.

El sitio ha sido desarrollado utilizando HTML y CSS, siguiendo una metodología SCRUM para la organización de tareas, asignación de roles y gestión del proyecto mediante GitHub Projects.

## Objetivos

* Informar sobre los conceptos básicos de la Inteligencia Artificial.
* Presentar aplicaciones actuales de la IA en diferentes sectores.
* Analizar ventajas, desafíos y tendencias futuras de esta tecnología.
* Aplicar buenas prácticas de desarrollo web y trabajo colaborativo.

## Tecnologías Utilizadas

* HTML5
* CSS3
* Git
* GitHub
* GitHub Projects
* Metodología SCRUM
